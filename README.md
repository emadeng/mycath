# Paltalk-like Chat (MVP)

This is a minimal prototype of a realtime chat server (Node.js + Socket.IO) and a static web client.
It is intended as a starting point for a Paltalk-style app (text/rooms). No database or authentication
beyond a simple username field is included â€” add JWT, DB, moderation, rate-limiting, HTTPS, etc.

## Run locally

1. Install Node.js (v18+ recommended).
2. In the project folder run:
   ```
   npm install
   npm start
   ```
3. Open `http://localhost:3000` in your browser.

## Files

- server.js       - Node.js + Socket.IO server
- package.json    - dependencies and start script
- client/index.html - static client (connects via socket.io)
